# YoutubeSearchDemo
youtube search demo using ionic

## Installation
```shell
$ bower install
```
## Bower Plugins
* angular-youtube-mb
* angular-translate

## Usage
set up API_KEY in www/modules/youtube/youtube.constant.js
```shell
$ ionic serve --lab
```

## Build Android APK
```shell
$ ionic platform add android
$ ionic resources --icon && ionic resources --splash
$ ionic build
```
